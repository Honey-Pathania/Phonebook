1. Open File.
2. Right Click and select "Open with Code" for opening VScode on that location.
3. open Command Prompt Terminal in VScode
4. Type "virtualenv venv"
4. Type "cd venv/scripts" then press Enter.
5. Type "activate".
6. Type "cd.." do this 2 times to be on the main directory.
7. Type "pip install -r requirements.txt"
8. Type "py manage.py runserver"
9. Open this link on browser   http://127.0.0.1:8000/
